## git 생성 후 Push 하기

1. 프로젝트 생성
2. github에서 repository 생성
3. 프로젝트 상위 폴더에서 git init
4. git add .
5. git commit -m "(커밋 메시지 작성)"
6. git remote add origin (2번에서 생성한 repository 주소)
7. git push origin master
