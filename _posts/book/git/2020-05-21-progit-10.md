# Pro git - 되돌리기

### 1. 되돌리기

* 커밋한 내용을 다시 커밋하고 싶으면 파일 수정 작업을 하고 staging Area에 추가한 다음 --amend 옵션을 사용하여 커밋을 재작성할 수 있다.

```bash
$ git commit --amend
$ git commit --amend -m "여기에 커밋 수정 메시지"
```

* 커밋 후에 수정한 것이 없고 커밋 메시지만 수정하고 싶으면 사용하면 됨
