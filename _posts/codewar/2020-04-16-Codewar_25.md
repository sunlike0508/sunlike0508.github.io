#### [Take a Ten Minute Walk](https://www.codewars.com/kata/54da539698b8a2ad76000228)

```java
public static boolean isValid(char[] walk) {

    if(walk.length != 10) {
        return false;
    }

    int westEast = 0;
    int northSout = 0;

    for(char block : walk) {
        if(block == 'w') {
            westEast++;
        } else if(block == 'e') {
            westEast--;
        } else if(block == 'n') {
            northSout++;
        } else {
            northSout--;
        }
    }

    return westEast == 0 && northSout == 0;
}
```

* 쉬운 문제였다. 설명할게 딱히 없다.
