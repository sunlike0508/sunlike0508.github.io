#### [Multiples of 3 or 5](https://www.codewars.com/kata/514b92a657cdc65150000006)



```java
public class Solution {

  public int solution(int number) {
    int sum = 0;
    
    for(int i = 3; i < number; i++) {
      
      if((i%3 == 0) || (i%5 == 0)) {
        sum+=i;
        
        System.out.println(i);
      }
    }
    
    return sum;
  }
}
```
