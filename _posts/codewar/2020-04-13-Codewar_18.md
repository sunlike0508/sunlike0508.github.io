#### [Bit Counting](https://www.codewars.com/kata/526571aae218b8ee490006f4)

```java
int count = 0;

do {
    if(n%2 == 1) {
        count++;
    }

    n = n/2;

} while(n > 0);

return count;

//return Integer.bitCount(n); 한 줄로 표현 가능;
```
