#### [Checking Groups](https://www.codewars.com/kata/54b80308488cb6cd31000161)

```java
public class Groups{

    public static boolean groupCheck(String s){
        String beforeS = "";

        do {
            beforeS = s;
            s = s.replaceAll("(\\(\\))|(\\{\\})|(\\[\\])", "");
        }while(beforeS.length() > s.length());

        return s.length() == 0;
    }
}
```

* 이건 이전에 나온 문제랑 같다. 
* 딱히 볼게 없었다.
