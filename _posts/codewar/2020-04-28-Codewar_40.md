#### [Detect Pangram](https://www.codewars.com/kata/545cedaa9943f7fe7b000048)

```java
public boolean check(String sentence) {

    for(int i = 0 ; i < 26; i++) {
        if(!sentence.toLowerCase().contains(String.valueOf((char)('a'+i)))){
            return false;
        }
    }

    return true;
}
```
